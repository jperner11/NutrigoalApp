# Treno Brand Guide

Treno means training. The identity is direct, compact, and built for repeated use inside product UI: a graphite foundation, bone text, and an acid-lime momentum mark.

## Logo

- Primary lockup: `apps/web/public/treno-logo.svg`
- Inverse lockup: `apps/web/public/treno-logo-inverse.svg`
- Compact mark: `apps/web/public/treno-mark.svg`
- Web component: `apps/web/src/components/brand/BrandLogo.tsx`
- Mobile component: `apps/mobile/src/components/BrandLogo.tsx`

The mark is a simplified **T** with a forward rail. Use the full lockup for marketing and empty states. Use the compact mark for app chrome, favicons, icons, and narrow navigation.

## Palette

- Graphite: `#0e1116`
- Elevated graphite: `#171b22`
- Bone: `#f5f7fa`
- Muted steel: `#8a8f97`
- Acid lime: `#cdf24e`
- Ink on accent: `#0a0a0a`

## Voice

Treno should sound coach-literate and plain-spoken: short sentences, concrete claims, no cute fitness hype.

Tagline: **One fitness app.**
