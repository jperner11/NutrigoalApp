# Fresh Account QA Checklist

## Personal Trainer flow
- Sign up as Personal Trainer
- Confirm PT dashboard appears
- Confirm PT navigation appears
- Send invite
- Confirm pending invite state
- Resend invite
- Cancel invite

## Client flow
- Open invite as new user
- Sign up
- Accept invite
- Confirm trainer-connected state
- Confirm no-plan-yet empty state if nothing assigned

## Plan workflow
- Trainer assigns diet plan
- Client sees diet plan
- Trainer assigns training plan
- Client sees training plan
- Managed client cannot create independent plans

## Communication workflow
- Trainer sends message
- Client receives and replies
- Trainer sends feedback request
- Client completes feedback request

## Safety and trust
- Privacy / Terms / FAQ / Support pages load
- Support links open correctly
- Account deletion path is visible
